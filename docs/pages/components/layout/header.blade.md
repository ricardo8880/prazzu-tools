# Cabeçalho principal

## Objetivo
Exibir a identidade do Prazzu Tools, a navegação principal, o controle de tema e as ações de autenticação.

## Funcionamento
- Cada item superior é um link direto e independente.
- **Recursos** direciona para a Central de Recursos e não utiliza seta ou dropdown.
- O item correspondente à rota atual recebe estado ativo e `aria-current`.
- Em telas menores, a navegação é disponibilizada pelo offcanvas existente.

## Regras visuais
- Deve seguir as mesmas cores, espaçamentos e estados dos demais itens do menu.
- Não criar indicadores de dropdown quando não existir menu suspenso funcional.
- Ações principais devem utilizar os componentes e variáveis roxas do projeto.
