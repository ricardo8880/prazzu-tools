# Calculadora de Simples Nacional

Módulo fiscal da Prazzu Tools.

## Lote 1

- registro do módulo e rota inicial;
- estrutura de domínio para anexos e faixas;
- catálogo interno de recursos gratuitos e Plus;
- página inicial estrutural usando Bootstrap;
- testes básicos de registro, níveis e faixa tributária.

A cobrança recorrente não é implementada neste lote. A separação dos recursos já possui identificadores estáveis para futura integração com o controle central de acesso.
